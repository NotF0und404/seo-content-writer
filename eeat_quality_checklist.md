# EEAT 與 SEO 內容品質檢查清單

使用本清單審核 SEO 內容方向、長尾詞文章、落地頁、指南、比較文與評測文。若任一高風險項為 Fail，必須先修改內容再交付。

## 1. 合規底線

| 檢查項 | Pass / Warning / Fail | 修正要求 |
|---|---|---|
| 不誇大、不保證結果、不承諾收益 | [ ] | 移除「保證、一定、穩賺、無風險、最強」等絕對化說法 |
| 不誘導高風險行為 | [ ] | CTA 必須低壓力、資訊導向，不可催促用戶立即高風險決策 |
| YMYL 主題有風險提示 | [ ] | 金融、投資、醫療、法律、安全內容需加入限制與非專業建議聲明 |
| 不捏造來源與數據 | [ ] | 每個資料點都能回溯到可訪問來源 |
| 不把不確定內容寫成確定事實 | [ ] | 使用「可能、取決於、公開資料顯示、仍需驗證」等保守表述 |
